====================================================
        AI CV Maker Professional - Standalone Version
====================================================

Aplikasi AI CV Maker Professional telah ditingkatkan menjadi perangkat lunak Desktop Murni (Standalone Software). Aplikasi ini sekarang berdiri sendiri seperti Microsoft Word atau Adobe Photoshop tanpa bergantung pada peramban web (browser).

----------------------------------------------------
FITUR DESKTOP BARU:
----------------------------------------------------
1. Format File Ekstensi ".cvm"
   Anda sekarang dapat menyimpan proyek CV Anda ke dalam bentuk file asli dengan menekan Ctrl+S. File ini berekstensi .cvm (CV Maker). Anda dapat membukanya kembali dari komputer Anda layaknya membuka file dokumen biasa.
   
2. Menu Bar Native & Shortcut
   Gunakan menu bagian atas OS Anda (File, Edit, View, Window) untuk membuat proyek baru, menyimpan, membuka, atau melakukan ekspor PDF. Shortcut keyboard (Ctrl+N, Ctrl+S, Ctrl+O, Ctrl+Z) semuanya didukung penuh.

3. Drag-and-Drop Foto Profil
   Sekarang Anda dapat menyeret (drag) sebuah foto dari File Explorer dan melepaskannya (drop) langsung ke atas ikon bulat profil di aplikasi tanpa perlu mengklik tombol Upload.

4. Cetak (Print) Langsung
   Terdapat tombol "Cetak" yang akan memunculkan dialog Print sistem Windows/Mac secara langsung untuk mengirim dokumen Anda ke Printer fisik.

5. Auto-Recovery (Pemulihan Otomatis)
   Selain menyimpan secara fisik ke file .cvm, aplikasi tetap mencadangkan pekerjaan terakhir Anda di belakang layar. Jika aplikasi tertutup mendadak, pekerjaan terakhir masih akan otomatis termuat.

----------------------------------------------------
LANGKAH PENGGUNAAN (CARA MEMULAI):
----------------------------------------------------

1. MENGATUR API KEY (Opsional untuk fitur AI)
   - Buka Aplikasi AI CV Maker.
   - Klik tombol "Pengaturan" (ikon Gear di pojok kanan atas).
   - Masukkan kunci "Gemini API Key" dari Google AI Studio.

2. MEMBUAT DAN MENYIMPAN PROYEK (.CVM)
   - Untuk membuat dokumen baru, klik "File > New Project" di Menu Bar atas.
   - Ketik data Anda. Untuk menyimpan kemajuan Anda sebagai file di komputer, klik "File > Save" atau tekan Ctrl+S.
   - Pilih direktori tempat Anda ingin menyimpannya.

3. MENGUBAH TAMPILAN & TEMPLATE
   - Di panel Kanan (Preview), gunakan menu melayang (floating menu) di atas kertas CV.
   - Pilih "Template" atau klik pada lingkaran warna untuk menyesuaikan warna.

4. MENGEKSPOR / MENCETAK CV
   - Klik "Ekspor CV PDF" untuk menjadikan file PDF.
   - Atau klik "Cetak" (Ikon Printer) jika Anda ingin langsung menge-print dari printer Anda.
